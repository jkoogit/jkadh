# 16.참고

이 폴더는 JKADH의 용어사전, 참고자료, 공통 참조 정보를 관리한다.

## 포함 문서

| 문서 ID | 문서명 | 상태 |
|---|---|---|
| REF-001 | [AI서비스플랫폼 용어사전](./REF-001_AI서비스플랫폼_용어사전.md) | Draft |

## 포함 예정 문서

현재 예정 문서는 없다.

## 관리 원칙

- 여러 문서에서 반복 사용하는 용어는 참고 문서로 분리한다.
- 개별 제품에만 적용되는 도메인 용어는 해당 제품 문서에서 관리할 수 있다.
- 공통 용어와 제품별 용어가 충돌하면 공통 용어의 의미를 먼저 확인한다.
- 참고 문서는 정책을 직접 강제하지 않고, 다른 문서가 같은 언어를 쓰도록 돕는다.
