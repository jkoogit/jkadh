# Backlog 미해결 인덱스

이 문서는 아직 해결되지 않은 백로그만 관리한다.

Resolved 또는 Rejected 상태가 된 백로그는 이 인덱스에서 제거하고, 개별 백로그 파일의 상태와 연결 이력만 갱신한다.

## 상태

| 상태 | 의미 | 처리 기준 |
|---|---|---|
| Open | 등록만 되었고 아직 분류하지 않음 | Issue 종료 시 정리한다. |
| Ready | 바로 Issue로 승격 가능 | 다음 Issue 선정 시 후보로 검토한다. |
| Blocked | 선행 Issue, 결정, 문서가 필요함 | 의존 대상 해결 후 재검토한다. |
| Deferred | 필요하지만 당장 우선순위가 낮음 | 정기 점검 시 재검토한다. |
| Issue Linked | GitHub Issue로 승격됨 | 연결 Issue와 PR 상태를 추적한다. |
| Resolved | 해결됨 | 미해결 인덱스에서 제거한다. |
| Rejected | 채택하지 않음 | 미해결 인덱스에서 제거한다. |

## 처리시점

| 처리시점 | 의미 |
|---|---|
| 즉시 | 현재 Issue 범위를 해치지 않고 바로 처리 가능 |
| Issue 종료 시 | 현재 Issue 완료 전 정리 필요 |
| 다음 Issue 선정 시 | 다음 작업 후보로 검토 |
| 선행 작업 완료 후 | 의존 대상이 해결되면 재검토 |
| 정기 점검 시 | 장기 보류 항목 점검 |
