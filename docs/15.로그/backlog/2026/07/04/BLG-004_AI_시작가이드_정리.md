## 1. 내용

AI가 JKADH 세션을 시작할 때 읽고 확인해야 하는 절차를 시작 문서로 정리한다.

## 2. 발생 배경

세션 시작 시 원격 dev, stg, main 상태 확인, 작업 브랜치 기준, 문서 읽기 순서가 필요하다는 요구가 있었다.

## 3. 기대 효과

- 세션이 바뀌어도 동일한 시작 절차를 따른다.
- 작업 브랜치와 기준 브랜치를 혼동하지 않는다.
- AI가 먼저 읽어야 할 문서 순서를 고정한다.

## 4. 처리 기준

Issue로 승격할 때 `STA-002_AI_시작가이드.md` 작성을 범위로 한다.
