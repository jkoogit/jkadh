## 1. 내용

폴더별 `README.md` 인덱스의 표 구조와 관련 문서 섹션을 표준화할지 검토한다.

## 2. 발생 배경

폴더별 README는 문서 본문이 아니라 인덱스 역할을 한다.

공식 문서 Header 템플릿을 적용하기보다, 인덱스 표와 관련 문서 연결 방식을 별도로 점검하는 것이 적절하다.

## 3. 기대 효과

- 폴더 인덱스에서 문서 ID, 문서명, 상태를 일관되게 확인할 수 있다.
- 공식 문서 템플릿과 README 인덱스 구조가 섞이지 않는다.

## 4. 처리 기준

- 공식 문서 Header 전체 현행화 이후 검토한다.
- 폴더 README의 기본 표 컬럼을 확인한다.
- 루트 `docs/README.md`는 별도 인덱스 문서로 볼지 함께 판단한다.
