## 1. 내용

Harness의 자율성을 제안, 작업 배치, Low-risk 수용, PR 루프 운영, 제한적 자동 승급 같은 단계로 정리한다.

## 2. 발생 배경

PRJ-002는 Phase 1부터 Phase 3까지의 현실 목표를 우선 정의했다. Harness 자율성 단계는 장기 로드맵과 연결되므로 별도 정리가 필요하다.

## 3. 기대 효과

- 자동화 범위를 한 번에 키우지 않고 단계적으로 검증할 수 있다.
- 각 단계에서 필요한 권한, 검증, 승인 조건을 연결할 수 있다.
- 완전 자율 개발 조직을 비목표로 유지하면서 실용적 자동화 수준을 정의할 수 있다.

## 4. 처리 기준

로드맵 문서에서 Harness 자율성 단계를 정의하고, 각 단계의 진입 조건과 금지 조건을 정리한다.
