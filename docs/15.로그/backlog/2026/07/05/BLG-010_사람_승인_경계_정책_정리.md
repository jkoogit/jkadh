## 1. 내용

Harness가 자동으로 수용하거나 진행할 수 없는 사람 승인 필수 조건을 정책으로 정리한다.

## 2. 발생 배경

PRJ-002는 사람 리드가 최종 승인권과 고위험 작업 허용권을 유지한다고 정의했다. 실제 운영에는 승인 필수 조건과 예외 처리 기준이 필요하다.

## 3. 기대 효과

- 고위험 작업의 자동 실행을 방지한다.
- 사람 리드의 승인권과 Harness의 자동화 범위를 분리한다.
- 정책 변경, 배포, Secret 접근, main 영향 작업의 처리 기준을 명확히 한다.

## 4. 처리 기준

정책 문서에서 승인 필수 조건, 승인 기록 방식, 예외 판단 기준, 승인 없이 중단해야 하는 조건을 정의한다.
