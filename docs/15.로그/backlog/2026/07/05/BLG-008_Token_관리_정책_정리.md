## 1. 내용

Token Budget과 Token Usage를 작업 지속, 중단, 축소, 분할, 재배치 판단에 사용하는 운영 정책을 정리한다.

## 2. 발생 배경

PRJ-002는 Token 기반 통제를 핵심 목표로 정의했다. 실제 운영에는 예산 단위, 사용량 기록, 초과 시 조치 기준이 필요하다.

## 3. 기대 효과

- 토큰과 비용을 작업 운영의 물리 제약으로 다룰 수 있다.
- 작업 중단, 축소, 분할, 재배치 기준을 명확히 할 수 있다.
- Report Loop에서 토큰 사용량을 일관되게 보고할 수 있다.

## 4. 처리 기준

`OPS-001 Token 운영` 문서에서 Token Budget, Token Usage, 잔여량 판단, 초과 대응, 보고 항목을 정의한다.
