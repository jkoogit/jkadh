# 15.로그

이 폴더는 작업 중 발생한 백로그, 기록, 로그성 항목을 관리한다.

## 하위 영역

| 경로 | 목적 |
|---|---|
| [backlog](./backlog/README.md) | 미해결 후속 작업 후보와 보류 항목 관리 |

## 운영 원칙

- 로그 항목은 생성일 기준 위치에 고정한다.
- 해결 여부에 따라 파일을 이동하지 않는다.
- 미해결 백로그만 `backlog/README.md`에 표시한다.
- 해결된 백로그는 개별 파일 상태를 갱신하고 `backlog/README.md`에서 제거한다.
