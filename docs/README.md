## 3. 인덱스 관리 기준

- 영역별 README는 `포함 문서`, `포함 예정 문서`, `관련 문서`를 기본 구조로 사용한다.
- 실제 파일이 있는 문서는 `포함 문서`에 둔다.
- 아직 파일이 없는 후보는 `포함 예정 문서`에 둔다.
- Backlog 인덱스는 상태와 처리시점을 함께 관리하므로 별도 구조를 유지한다.
